# Christine Technologies Website

Static website for christinetechnologies.com.

## Deploy to GitHub Pages
1. Create a GitHub repository.
2. Upload these files to the repository root.
3. Go to Settings → Pages.
4. Select Deploy from a branch, main, root.
5. Add `christinetechnologies.com` as the custom domain.

## Deploy to Cloudflare Pages
1. Create a GitHub repository and upload these files.
2. Cloudflare → Workers & Pages → Create application → Pages.
3. Connect GitHub repo.
4. Build command: blank. Output directory: `/`.
5. Add custom domain.

## Before Publishing
Update privacy/terms effective dates, support email, and add your Google Play link when ready.
